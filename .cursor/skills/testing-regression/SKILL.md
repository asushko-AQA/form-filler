# Regression Testing for Core Form Filling

## When to use this skill

Use this skill when:

- Changes affect **content.js** or core filling logic.
- The user requests a **regression checklist** around filling behavior.

## Scope

- Phone country code dropdowns and select-like widgets.
- Delayed/staggered filling and SPA/validation flows.
- Two-pass custom selector filling.

For detailed steps, consult the **Regression Checklist – Form Filling Core** section in `TESTS.md`.
